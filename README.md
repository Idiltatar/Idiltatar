## Featured Projects

### AlertIQ

AI-powered alert classification platform built with Python, Flask, scikit-learn and SQLite.

- Classifies monitoring alerts as `Critical` or `Noise`.
- Uses engineered alert features and a Logistic Regression model.
- Includes REST APIs, a dashboard, confidence scores and low-confidence review support.

Repository: [Idiltatar/alertiq](https://github.com/Idiltatar/alertiq)

### Mini Jira

Issue tracking system built with Ruby on Rails and PostgreSQL.

- Supports projects, users, tickets, assignments and workflow tracking.
- Uses Devise authentication and a relational database model.
- Demonstrates Rails CRUD workflows and project-management functionality.

Repository: [Idiltatar/mini_jira](https://github.com/Idiltatar/mini_jira)

### Secure Blog Project

Flask security learning project for reviewing secure and insecure web application patterns.

Repository: [Idiltatar/secure-blog-project](https://github.com/Idiltatar/secure-blog-project)
